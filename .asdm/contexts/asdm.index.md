# Nice Today App - ASDM Context Index

## Overview
The Nice Today App is a comprehensive mobile application built with React and Capacitor, focusing on biorythms, astrology, health tracking, and personal insights. The app integrates various traditional Chinese fortune-telling systems with modern wellness tracking features.

## Project Information
- **Name**: Nice Today App (生物节律助手)
- **Version**: 1.0.12
- **Build Time**: 2023-12-27T00:00:00Z
- **Git Commit**: main
- **Main Features**: Biorythm Analysis, Constellation Fortune, Mayan Calendar, MBTI Tests, Life Trend Analysis

## Key Components
- **Frontend**: React-based application with modular architecture
- **Mobile Platform**: Capacitor for Android/iOS deployment
- **State Management**: React Context API and custom hooks
- **UI Framework**: Tailwind CSS with responsive design
- **Data Storage**: Capacitor Preferences and local storage

## Architecture Layers
- **Presentation Layer**: React components with lazy loading
- **Business Logic**: Custom hooks and utility functions
- **Data Access**: Capacitor plugins and local APIs
- **Storage Layer**: Local storage and Capacitor Filesystem

## Available Routes
The application provides extensive routing for various features including biorhythm tracking, horoscope readings, Bazi analysis, Zi Wei analysis, and more.

## Technology Stack
- React 18.2.0
- React Router DOM 7.10.1
- Capacitor 5.7.8
- Tailwind CSS
- Chart.js for data visualization
- Lunar-JavaScript for calendar calculations
